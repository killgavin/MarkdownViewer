## 6. 程式碼測試

### 行內程式碼

使用 `Console.WriteLine()` 來輸出訊息。

這是一個 `變數` 的範例。

### 程式碼區塊

```vb
Public Class HelloWorld
    Public Shared Sub Main()
        Console.WriteLine("你好，世界！")
        Dim message As String = "Markdown 檢視器"
        Console.WriteLine(message)
    End Sub
End Class
```

```python
def hello_world():
    print("Hello, World!")
    message = "Markdown Viewer"
    print(message)

if __name__ == "__main__":
    hello_world()
```

```javascript
function helloWorld() {
    console.log("Hello, World!");
    const message = "Markdown Viewer";
    console.log(message);
}

helloWorld();
```
