## 技術規格

### 開發環境
- **.NET SDK：** 10.0
- **開發語言：** Visual Basic .NET (VB.NET)
- **UI 框架：** Windows Forms
- **目標平台：** Windows 10/11 (x64, x86, ARM64)

### 相依套件
| 套件名稱 | 版本 | 用途 |
|---------|------|------|
| Markdig | 0.37.0 | Markdown 轉 HTML 引擎 |

### 核心功能
1. **檔案開啟**
   - 支援 .md 和 .markdown 檔案
   - 檔案大小限制：10 MB
   - UTF-8 編碼支援

2. **Markdown 渲染**
   - 完整支援標準 Markdown 語法
   - 支援擴展語法（表格、任務清單等）
   - 自訂 CSS 樣式
   - 微軟正黑體字型

3. **使用者介面**
   - 工具列（開啟檔案、關於）
   - 主要預覽區（WebBrowser 控制項）
   - 狀態列（檔案資訊顯示）
   - 正體中文介面
