## 簡介

Markdown 檢視器是一個簡單易用的桌面應用程式，專為檢視 Markdown 格式文件而設計。本軟體使用 VB.NET Windows Forms 技術開發，提供完整的正體中文介面，讓您能夠輕鬆開啟並預覽 Markdown 文件。

### 主要特色

- ✅ **簡單易用** - 直覺的操作介面，一鍵開啟檔案
- ✅ **完整支援** - 支援所有標準 Markdown 語法
- ✅ **美觀顯示** - 使用微軟正黑體字型，提供舒適的閱讀體驗
- ✅ **正體中文** - 所有介面文字均使用正體中文
- ✅ **快速渲染** - 即時轉換並顯示 Markdown 內容

## 系統需求

### 最低需求

- **作業系統：** Windows 10 (64-bit) 或更新版本
- **記憶體：** 2 GB RAM
- **硬碟空間：** 100 MB 可用空間
- **.NET Runtime：** .NET 10.0 Runtime

### 建議需求

- **作業系統：** Windows 11 (64-bit)
- **記憶體：** 4 GB RAM 或更多
- **硬碟空間：** 200 MB 可用空間

## 安裝說明

### 步驟一：安裝 .NET Runtime

1. 前往 Microsoft .NET 官方網站下載 .NET 10.0 Runtime
2. 執行下載的安裝程式
3. 依照安裝精靈的指示完成安裝

### 步驟二：執行 Markdown 檢視器

1. 下載 Markdown 檢視器程式檔案
2. 解壓縮至任意資料夾
3. 雙擊 `MarkdownViewer.exe` 啟動程式

## 使用說明

### 基本操作

#### 開啟 Markdown 檔案

1. 啟動 Markdown 檢視器
2. 點選工具列上的 **「開啟檔案」** 按鈕
3. 在檔案選擇對話框中瀏覽並選擇您要檢視的 Markdown 檔案
   - 支援的檔案格式：`.md`、`.markdown`
4. 點選 **「開啟」** 按鈕
5. 檔案內容將立即顯示在視窗中

#### 檢視說明

- **捲動瀏覽：** 使用滑鼠滾輪或視窗右側的捲動軸來瀏覽文件內容
- **調整視窗大小：** 可以自由調整視窗大小以符合您的螢幕配置
- **狀態列訊息：** 視窗底部的狀態列會顯示當前載入的檔案名稱

#### 檢視關於資訊

1. 點選工具列上的 **「關於」** 按鈕
2. 查看軟體版本與相關資訊
3. 點選 **「確定」** 關閉對話框

### 支援的 Markdown 語法

Markdown 檢視器支援完整的 Markdown 語法，包括：

#### 基本語法

- **標題：** `#` 到 `######` 六個層級
- **粗體：** `**粗體文字**` 或 `__粗體文字__`
- **斜體：** `*斜體文字*` 或 `_斜體文字_`
- **刪除線：** `~~刪除文字~~`
- **行內程式碼：** `` `程式碼` ``

#### 清單

```markdown
- 無序清單項目 1
- 無序清單項目 2

1. 有序清單項目 1
2. 有序清單項目 2
```

#### 連結與圖片

```markdown
[連結文字](https://example.com)
![圖片替代文字](image.png)
```

#### 程式碼區塊

````markdown
```python
def hello():
    print("Hello, World!")
```
````

#### 引用

```markdown
> 這是引用文字
> 可以有多行
```

#### 表格

```markdown
| 欄位 1 | 欄位 2 | 欄位 3 |
|--------|--------|--------|
| 資料 1 | 資料 2 | 資料 3 |
```

#### 水平線

```markdown
---
或
***
```

## 常見問題

### Q1: 程式無法啟動，顯示「找不到 .NET Runtime」錯誤

**A:** 請確認已安裝 .NET 10.0 Runtime。您可以從 Microsoft 官方網站下載並安裝。

### Q2: 開啟檔案後沒有顯示內容

**A:** 請確認：
1. 檔案格式是否為 `.md` 或 `.markdown`
2. 檔案編碼是否為 UTF-8
3. 檔案內容是否為空白

### Q3: 中文字顯示不正常

**A:** Markdown 檢視器使用微軟正黑體作為預設字型。請確認：
1. 系統已安裝微軟正黑體字型
2. Markdown 檔案儲存時使用 UTF-8 編碼

### Q4: 程式碼區塊的語法高亮顯示不正常

**A:** 目前版本的程式碼區塊會以固定寬度字型顯示，但不包含語法高亮功能。此功能將在未來版本中加入。

### Q5: 可以同時開啟多個檔案嗎？

**A:** 目前版本一次只能開啟一個檔案。若要檢視其他檔案，請重新點選「開啟檔案」按鈕。多分頁功能將在未來版本中提供。

### Q6: 是否支援列印功能？

**A:** 目前版本尚未提供列印功能。您可以使用瀏覽器的列印功能作為替代方案。此功能將在未來版本中加入。

### Q7: 如何編輯 Markdown 檔案？

**A:** Markdown 檢視器是一個純檢視工具，不提供編輯功能。若需編輯，請使用文字編輯器（如 Notepad++、Visual Studio Code 等）。
