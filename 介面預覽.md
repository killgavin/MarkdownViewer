## 介面元件說明

### 1. 標題列
- **位置：** 視窗最上方
- **內容：** "Markdown 檢視器"
- **功能：** 顯示應用程式名稱，標準 Windows 視窗控制項（最小化、最大化、關閉）

### 2. 工具列（ToolStrip）
- **位置：** 標題列下方
- **高度：** 25 像素
- **按鈕：**
  - **開啟檔案** - 開啟 Markdown 檔案選擇對話框
  - **關於** - 顯示應用程式資訊

### 3. 主要預覽區（WebBrowser）
- **位置：** 工具列與狀態列之間
- **功能：** 顯示轉換後的 HTML 內容
- **樣式：**
  - 字型：微軟正黑體
  - 內距：20px
  - 背景：白色
  - 支援捲動

### 4. 狀態列（StatusStrip）
- **位置：** 視窗最下方
- **高度：** 22 像素
- **內容：** 顯示當前檔案名稱或狀態訊息

- **背景色：** 白色 (#FFFFFF)

## Windows 執行環境預覽

在 Windows 環境中執行時，應用程式會具有標準的 Windows Forms 外觀：

- **視窗框架：** 標準 Windows 視窗框架（Aero/Modern 樣式）
- **按鈕樣式：** 符合作業系統主題
- **字型渲染：** ClearType 字型平滑化
- **捲軸樣式：** 系統預設捲軸樣式

---

## 操作提示

### 鍵盤操作
- **Alt + F4** - 關閉視窗
- **滑鼠滾輪** - 捲動內容
- **Ctrl + 滾輪** - 縮放內容（WebBrowser 支援）

### 滑鼠操作
- **左鍵單擊** - 點選按鈕、連結
- **右鍵單擊** - WebBrowser 右鍵選單（複製、列印等）
- **拖曳視窗邊緣** - 調整視窗大小

---

**注意：** 此為介面設計模擬圖，實際執行時的外觀會依據 Windows 作業系統版本與主題設定而有所不同。
