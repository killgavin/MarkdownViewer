# Markdown 檢視器範例

## 歡迎使用 Markdown 檢視器

這是一個使用 VB.NET Windows Forms 開發的 Markdown 文件檢視軟體。

## 主要功能

- ✅ 開啟並顯示 Markdown 檔案
- ✅ 支援標準 Markdown 語法
- ✅ 支援程式碼高亮顯示
- ✅ 支援表格顯示
- ✅ 正體中文介面

## 支援的 Markdown 語法

### 標題

使用 `#` 符號建立標題，從 `#` 到 `######` 共六個層級。

### 清單

#### 無序清單

- 項目 1
- 項目 2
  - 子項目 2.1
  - 子項目 2.2
- 項目 3

#### 有序清單

1. 第一項
2. 第二項
3. 第三項

### 程式碼

行內程式碼使用 `反引號` 包圍。

程式碼區塊範例：

```vb
Public Class HelloWorld
    Public Shared Sub Main()
        Console.WriteLine("你好，世界！")
    End Sub
End Class
```

### 表格

| 功能 | 說明 | 狀態 |
|------|------|------|
| 開啟檔案 | 載入 Markdown 檔案 | ✅ 完成 |
| 預覽顯示 | HTML 渲染顯示 | ✅ 完成 |
| 語法支援 | 完整 Markdown 語法 | ✅ 完成 |

### 引用

> 這是一個引用區塊。
> 可以用來強調重要的內容。

### 強調

- **粗體文字**
- *斜體文字*
- ~~刪除線~~

### 連結與圖片

[GitHub](https://github.com)

## 使用說明

1. 點選「開啟檔案」按鈕
2. 選擇要檢視的 Markdown 檔案（.md 或 .markdown）
3. 檔案內容會自動轉換成 HTML 並顯示

## 技術規格

- 開發語言：VB.NET
- 框架：Windows Forms (.NET 10.0)
- Markdown 引擎：Markdig
- 介面語言：正體中文
